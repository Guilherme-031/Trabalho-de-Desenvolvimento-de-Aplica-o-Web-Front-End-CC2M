Descrição do produto/serviço:
O projeto consiste no desenvolvimento de um site voltado para uma loja de tênis, com foco em oferecer um ambiente dinâmico, visualmente atrativo e acessível para pessoas de todas as idades. O site contém páginas de Home, Produtos com detalhamento, Sobre a Empresa e uma página de Compra com carrinho funcional.

Tecnologias utilizadas:
Foram utilizadas as seguintes tecnologias:
HTML para estruturação das páginas,
CSS para estilização e identidade visual,
JavaScript para interatividade e dinamismo,
Bootstrap para criação de layouts responsivos e modernos.
